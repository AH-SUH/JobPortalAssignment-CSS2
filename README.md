# Job Portal CSS Assignment

This project is a recreation of a Job Portal webpage using HTML and CSS, based on a provided design mockup.

## 📌 Project Overview

The goal of this assignment was to practice building a structured webpage layout and styling it to closely match a reference image.

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox & Grid for layout

## ✨ Features

- Responsive navigation bar
- Hero section with job search inputs
- Styled "Popular Job Searches" tags
- Login and Registration forms
- Multi-column footer with job links
